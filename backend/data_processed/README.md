This folder should host all the processed data from the raw data collected in data_raw folder

From Cache to parquet files

Notebooks under notebooks/Data Collection/ should be used to collect the data and save it in this folder as parquet files
Some notebooks  did not do this and just added to the cache (data_raw/cache) and the ML notebooks worked off them directly

This is not good practice and should be avoided

All data should be stored in this folder as parquet files and should be used by the ML notebooks


So Data_Collection Notebooks -> gets raw data stores in data_raw/cache -> converts to parquet files and stores in data_processed

ML Notebooks in the appropriate phases -> reads from data_processed and builds models
